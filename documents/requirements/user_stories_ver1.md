#### 1. As a user, I want to record activities by specifying the date, time (from/to), task, and a tag, so that I can keep track of what I worked on.
##### Acceptance Criteria:
- The user can input a date and time range for each task.
- The user can tag each activity.
- The system should validate that the 'time from' is earlier than the 'time to'.
- The user can edit or delete recorded activities.
#### 2. As a user, I want to query my recorded activities based on date, task, or tag, so that I can see I have used my time for specific activities or during specific periods.
##### Acceptance Criteria:
- The user can query activities based on:
    - A specific date or a date range.
    - A specific task.
    - One or multiple tags.
- The user should be able to sort the query results by date, duration.
- The user can view details such as total time spent on filtered tasks.
